The Building Block

It starts with
one module.

Every Precast Africa development begins with the same factory-built unit. Fully prefinished. Crane-set on site as a complete, ready-to-live-in home. From this single block, everything else scales.

Across

Lay them
side by side.

Connect the same module horizontally and the unit becomes a row of homes, a townhouse cluster, a family estate. The factory build doesn't change. The site does. One block, multiplied across the ground plane.

Up

Stack them
up to six.

Stack the same modules vertically and you have a walk-up, a mid-rise apartment block, a six-storey housing scheme. And these two directions, across and up, are only the beginning of what one module family can do.

Building Configurations

Any layout. One system.

From single-storey townhouse rows to six-storey mid-rise apartments, the same module family adapts to almost any residential typology.

Configuration

Single Storey
