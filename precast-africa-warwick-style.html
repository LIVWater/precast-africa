The Module Family

Three modules. Any configuration.

Module Type A

Main Module

Primary habitable space. Configurable for studio, 1-bed, and 2-bed layouts. Flexible window arrangements, door positions, and internal partition options within structural limits.

View module
Module Type B

Wet-Core Module

Centralises bathrooms, kitchens, plumbing, drainage, and electrical services. Factory-tested before dispatch. Eliminates the most common defect source in conventional residential builds.

View module
Module Type C

Stair / Core Module

Vertical circulation, stair access, structural core, and service shafts for multi-storey buildings. Provides lateral stability across the full stacked structure.

View module
